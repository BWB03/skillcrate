// how it works

Three steps. Zero cost.

Skills are just GitHub repos with a standardized structure. No vendor lock-in, no API keys, no subscriptions.

01

Find a skill

Browse the catalog or search by category — advertising, catalog management, inventory, compliance, and more.

02

Clone & configure

Each skill repo includes a SKILL.md with setup instructions for your agent platform of choice.

03

Run it

Load the skill into your agent and let it work. Customize anything — it's all open source, MIT licensed.
